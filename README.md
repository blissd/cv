# CV

My CV, beautifully rendered by `LaTex`.

## Dependencies

Install dependencies:

```bash
apt install texlive-base
apt install pandoc
```

## Generate CV in PDF

To build the `pdf` CV:

```bash
make
```

or:

```bash
make pdf
```

## Generate CV in MS docx
To build the 'docx' CV:

```bash
make doc
```
